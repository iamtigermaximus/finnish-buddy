## 📚 About

Finnish Buddy is a comprehensive language learning platform that helps users master Finnish through AI-generated lessons, interactive exercises, and personalized quizzes. The app covers all CEFR levels from A1 (Beginner) to C2 (Proficient).

### ✨ Features

- **🤖 AI-Powered Lessons** - Dynamic lesson generation based on your level and topic
- **📚 6 CEFR Levels** - From absolute beginner to mastery
- **🎯 Interactive Quizzes** - Test your knowledge with instant feedback
- **💡 Memory Aids** - Learn with mnemonics like UNO method for partitive case
- **📝 Practice Exercises** - Hands-on exercises with hints and feedback
- **🐻 Friendly Mascot** - Learn with Otso the bear
- **📱 Responsive Design** - Learn anywhere, on any device
- **🔐 User Authentication** - Secure accounts with NextAuth.js

## 🔧 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/finnish-buddy.git
cd finnish-buddy
2. Install dependencies
bash
npm install
3. Set up environment variables
Create a .env.local file in the root directory:

bash
# Database
DATABASE_URL="postgresql://postgres:password@localhost:5432/finnish_buddy"

# Authentication
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"

# AI Provider (DeepSeek)
DEEPSEEK_API_KEY="your-deepseek-api-key-here"
4. Set up the database
bash
# Push schema to database
npx prisma db push

# Seed initial data (levels)
npm run db:seed
5. Run the development server
bash
npm run dev
Open http://localhost:3000 to see the app.

Examples - See the concept in real Finnish sentences

Practice - Interactive exercises with hints

Quiz - Test your knowledge with scoring

AI-Generated Content
Every lesson is dynamically generated by DeepSeek AI based on:

The specific topic (Personal Pronouns, Partitive Case, etc.)

The user's CEFR level (A1-C2)

Previous performance (for personalized quizzes)

Memory Aids
Each lesson includes custom mnemonics like:

UNO method for Partitive case (Uncountable, Negative, Ongoing)

SALE MALTA for location cases

My Sister Has My Two Hands for personal pronouns
```
